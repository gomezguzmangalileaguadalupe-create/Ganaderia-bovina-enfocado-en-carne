Problema Prototípico

El sector agropecuario mexicano representa uno de los componentes más relevantes para la estabilidad económica, la seguridad alimentaria y el desarrollo social del país. Dentro de este contexto, la ganadería bovina destinada a la producción de carne ocupa una posición estratégica debido a su capacidad para generar empleo, fortalecer las economías rurales y abastecer tanto al mercado nacional como internacional. Sin embargo, los pequeños y medianos productores pecuarios enfrentan un entorno altamente competitivo y vulnerable, caracterizado por la volatilidad de los precios, el incremento constante en los costos de producción y la dependencia de cadenas de intermediación que reducen significativamente sus márgenes de ganancia.

Mercado Justo nace como una propuesta tecnológica y analítica desarrollada para contrarrestar estas problemáticas mediante el uso de herramientas avanzadas de Big Data, optimización de rutas de transporte y modelado predictivo de mercados financieros pecuarios.

Metodología del Proyecto

Nuestra solución implementa un ciclo analítico completo: ingesta distribuida y limpieza masiva de datos históricos mediante PySpark, entrenamiento de modelos estadísticos y de Machine Learning (como Random Forest y modelos de series de tiempo) para pronosticar los precios del ganado, y modelado de grafos logísticos para trazar cadenas de distribución óptimas.

Conclusión del Estudio

En definitiva, la solución materializada a través del Dashboard interactivo en Gradio no sólo valida la viabilidad metodológica de este estudio, sino que democratiza el acceso a la tecnología. "Mercado Justo" se posiciona como un modelo de agronegocio próspero, replicable, inclusivo y sustentable, sentando un precedente de cómo la ciencia de datos puede ponerse al servicio de la justicia económica y el desarrollo rural de México.

import pandas as pd
from pyspark.sql import SparkSession

# Asegurar que SparkSession esté disponible y activa
# Se detiene cualquier sesión existente para evitar conflictos y se crea una nueva.
if 'spark' in locals() and spark.sparkContext._jsc is not None:
    spark.stop()

spark = SparkSession.builder \
    .appName("BigDataCarneBovina") \
    .getOrCreate()

print("SparkSession creada exitosamente.")

# Cargar el archivo CSV usando pandas
pd_df = pd.read_csv("//content/Base_14000_Datos.csv")

# Convertir el DataFrame de pandas a un DataFrame de PySpark
df = spark.createDataFrame(pd_df)

print("DataFrame de PySpark creado exitosamente desde pandas.")

print("\nCantidad de registros (desde pandas a PySpark):")
print(df.count())

print("\nEstructura de datos (desde pandas a PySpark):")
df.printSchema()

print("\nPrimeros registros (desde pandas a PySpark):")
df.show(5)

from pyspark.sql.types import DateType, DoubleType, IntegerType
from pyspark.sql.functions import regexp_replace, col, trim, when, lit, isnull, length

# Convertir la columna 'fecha' a tipo DateType
df = df.withColumn("fecha", df["fecha"].cast(DateType()))

# Limpieza y conversión de tipos para las columnas numéricas
df = df.withColumn("precio_por_kg", regexp_replace(col("precio_por_kg"), "[\\$,]", ""))
df = df.withColumn("precio_por_kg", col("precio_por_kg").cast(DoubleType()))

df = df.withColumn("volumen_toneladas", regexp_replace(col("volumen_toneladas"), "[,]", ""))
df = df.withColumn("volumen_toneladas", col("volumen_toneladas").cast(DoubleType()))

# Eliminar espacios en blanco en columnas de texto
df = df.withColumn("estado_origen", trim(col("estado_origen")))
df = df.withColumn("estado_destino", trim(col("estado_destino")))
df = df.withColumn("tipo_corte", trim(col("tipo_corte")))

print("\nEstructura después de la limpieza:")
df.printSchema()
df.show(5)

# --- Preparación para Modelado Temporal ---
# Convertir de vuelta a Pandas para análisis de series de tiempo local
df_pandas = df.toPandas()
df_pandas['fecha'] = pd.to_datetime(df_pandas['fecha'])
df_ts = df_pandas.groupby('fecha')['precio_por_kg'].mean().asfreq('D').fillna(method='ffill')

import matplotlib.pyplot as plt
df_ts.plot(figsize=(12,6), color='#1A237E', title="Evolución Histórica de Precios")
plt.grid(True, linestyle=':', alpha=0.6)
plt.show()

from statsmodels.tsa.statespace.sarimax import SARIMAX

# Definimos el orden regular (p,d,q) y el orden estacional (P,D,Q,s)
# Para datos diarios con ciclos semanales (s=7):
modelo_sarima = SARIMAX(df_ts, order=(5, 1, 0), seasonal_order=(1, 1, 1, 7))
resultados_sarima = modelo_sarima.fit()

# Predicción
pred_sarima = resultados_sarima.forecast(steps=30)
print("\nPredicciones SARIMA para los próximos 30 días:")
print(pred_sarima.head())

from prophet import Prophet

# Prophet requiere un DataFrame con columnas específicas: 'ds' y 'y'
df_prophet = df_ts.reset_index()
df_prophet.columns = ['ds', 'y']

m = Prophet(yearly_seasonality=True, weekly_seasonality=True)
m.fit(df_prophet)

future = m.make_future_dataframe(periods=30)
forecast = m.predict(future)
print("\nPredicciones con Prophet completadas.")
